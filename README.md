# Rspec playground

Running examples (ensure you have docker & docker-compose installed):

```bash
git clone git@github.com:dsalahutdinov/rspec_playground.git
make
```

## before callback

```bash
make rspec_before
```
## before(:all) callback

```bash
make rspec_before_all
```

## let(:some_var) test data definition

```bash
make rspec_let
```

## let!(some_var) test data definition

```bash
make rspec_let_bang
```
